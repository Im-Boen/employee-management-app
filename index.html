Employee Management System

Welcome to the Employee Management System — a lightweight web application built with HTML, CSS, and JavaScript. This system allows HR staff and administrators to manage employee contact records efficiently. Use the navigation bar above to add new contacts or browse the full employee directory. All data is stored locally in your browser for a seamless offline experience.
