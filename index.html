About This System

Employee Management System

Welcome to the Employee Management System — a lightweight HR web application built with HTML, CSS, and JavaScript. Designed for HR administrators to efficiently manage employee contact records. Use the navigation bar to add new contacts or browse the full employee directory. All data is persisted locally in your browser for a seamless offline experience.
